# Brew Haven Coffee Shop

## Project Overview

Brew Haven Coffee Shop is a responsive coffee shop website developed using HTML, CSS, and JavaScript. The website is designed to provide an attractive online presence for a coffee shop by showcasing its menu, services, customer reviews, and contact information through a modern and user-friendly interface.

The project focuses on responsive web design, interactive navigation, and visually appealing UI components to create a realistic coffee shop experience.

---

## Features

* Fully responsive design for desktop, tablet, and mobile devices
* Modern coffee shop landing page
* Interactive navigation menu
* Mobile-friendly hamburger menu
* Attractive hero section with call-to-action button
* Coffee menu showcase with images and pricing
* Features and services section
* Customer reviews section
* Contact information section
* Smooth scrolling navigation
* Clean and professional UI design
